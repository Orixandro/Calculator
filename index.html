Instructions

Buttons and displays

Click the buttons in the calculator's interface in order to execute their specific functions:

  • Number buttons: These buttons append the number clicked to the current operating value.
  • Operator buttons: These are the buttons colored in orange, they will set up their respective operation when clicked. In the case of the equal button, it will perform the current operation.
  • AC button: This button can be used to clear both the current number and current operation.
  • Delete button: This button can be used to delete the last number inputed.
  • Decimal button: This button can be used to separate the whole and decimal parts of a number.
  • Sign button: This button can be used to change the sign of the current number.

There are 2 displays in this calculator; the smaller display at the top will display the operation currently being performed. The bigger display below will only display the number currently being typed or the result of an operation.

History section

The history section of the page provides a list of the calculations performed.

Any calculation can be deleted by pressing the red "Remove" button that appears at the right side.

Additionally, you can copy the result of any operation to your clipboard by pressing the "Copy result" button.

You can delete all the operations performed by clicking the "Remove all" button.

Keyboard compatibility

It is possible to use the calculator only by pressing keys in your keybord. The following list contains every key that can perform each action:
